## **S3 ingestion** 

## **Supported environments (bucket name):** 

**S3 Ingestion (S3I)** is a scalable and efficient data ingestion solution leveraging AWS services, including S3, SNS, and SQS. 

Each supported environment is provisioned with a dedicated S3 bucket, granting write access to all AWS accounts within that environment. 

**Integration - s3://lama-integration-ingest US2 - s3://lama-awsus2-ingest EU2 - s3://lama-awseu2-ingest APJ1 - s3://lama-awsapj1-ingest PSCC - s3://lama-uspscc-ingest** 

While there are no strict requirements for data delivery methods, we recommend using FluentBit for optimal performance and reliability.

2.  Configure _fluent-bit.conf_ in the Host .  Please review the as  API Gateway Team's config as a reference - https://github. concur.com/gateway/api-gateway/blob/main/conf/env /integration/fluent-bit.conf 

3. Please stop Filebeat (if already enabled), as having both FluentBit & Filebeat agents will generate duplicates
