## **Getting Started - Logging Service** 

Concur Logging Service is a dial-tone Concur Cloud Service, providing an easy way to to collect data from any application, service, infrastructure, and search, analyze, and visualize the data in real time. 

The service is developed, configured, deployed, hosted and operated by dedicated E2E team.  Service users (R&D teams, Service Management teams, Site Reliability Engineering teams, Customer Support, etc) use the logging solution as a service. 

Logging Service is based on Elasticsearch (No SQL, highly scalable distributed storage), Logstash (data in pipeline), Kibana (visualization platform) technology stack and provides a complete logging and visualization solution, plus automated alerting and security features. 

*NOTE - RabbitMQ is no longer supported
