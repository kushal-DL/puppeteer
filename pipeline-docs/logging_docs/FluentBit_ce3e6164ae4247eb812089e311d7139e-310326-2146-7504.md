## **FluentBit** 

FluentBit is an open source **Log Processor and Forwarder** which allows you to collect any data like metrics and logs from different sources, enrich them with filters and send them to multiple destinations. It's the preferred choice for containerized environments like Kubernetes. 

FluentBit is designed with performance in mind: high throughput with low CPU and Memory usage. It's written in **C** language and has a pluggable architecture supporting more than 70 extensions for inputs, filters and outputs.

## **FIPS** 

Linux vendor packages and Concur package `fluentbit-*. cnqr` are FIPS-compliant. Windows package is not FIPS-compliant, and is not recommended due to untrusted encryption library anyway. 

FluentBit is a CNCF (Cloud Native Computing Foundation) subproject under the umbrella of Fluentd.

## Windows Event Log 

```
fluent-bit.conf: |
[SERVICE]
        Flush         1
        Log_Level     warning
        Daemon        off
[INPUT]
    Name         winlog
    Channels     Setup,Windows PowerShell
[OUTPUT]
        Name                        http
        Match                       *
        Host
<FILL_ENV_ENDPOINT>
        Port                        443
        URI                         /
        Format                      json_lines
        tls                         On
        tls.verify                  Off
        json_date_key               @timestamp
        json_date_format            iso8601
        net.connect_timeout         15
        net.keepalive_idle_timeout  600
```
