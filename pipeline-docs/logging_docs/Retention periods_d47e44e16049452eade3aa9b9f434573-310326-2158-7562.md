## **Retention periods** 

## What is Log Retention Period? 

A log retention period is **the amount of time you keep logs** . Log retention refers to the archiving of event logs, particularly those related to security, GDPR concerning the duration for which you store these log entries. By keeping records, service & security teams can gain critical insight into the activities happening on their network. 

Retaining logs for extended periods is optimal for security and compliance measures. However, this can become costly. That is we have created log retention policies that outline set log retention periods. 

## Concur Log Retention Policy?
