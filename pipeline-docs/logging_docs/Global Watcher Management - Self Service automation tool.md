## **Global Watcher Management - Self Service automation tool** 

Project Feedback Current Availability 

With the closed CCPS environment, there was a need to have a fully automated Watcher Management process. This was designed purely and only for CCPS/CTE-3 needs. 

Based on users' experience, we took it to the next level, where users are able to manage their watches globally **across all** (eventually) environments, **usin g templates** ... 

## Project 

Fully codified solution. Take your watch, make a simple update, and let it deploy everywhere you need! Clear ownership model. 

In the future, this will be the ONLY way to manage your watches - the same as we have it now in the USPSCC environment. Any manual changes will be disabled.
