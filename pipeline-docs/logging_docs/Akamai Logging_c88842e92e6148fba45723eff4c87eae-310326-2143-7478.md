## **Akamai Logging** 

We collect logs from Akamai US, EMEA and China regions. 

These logs are very useful for tracking how users are hitting our normal web pages, API calls and image uploading.   These logs are a great way to track down users having issues connecting to our services.  You will be able to see when the user sent a request and if they are actually hitting our services. To understand Akamai logs format, read through Akamai Log Format . 

US and EMEA Akamai logs are currently collected into separate service hosted in AWS-PR environment. Find service endpoints on Service Endpoints - Logging Service . 

China Akamai logs are collected in same logging service hosted in BEIPR1, where application logs are collected, different index "akamai-*" is used. Find service endpoints on Service Endpoints - Logging Service .
