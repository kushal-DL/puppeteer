## **FunctionBeat** 

FunctionBeat is an Elastic Beat that you deploy as a function in your serverless environment to collect data from cloud services and ship it to the Elastic Stack. 

Version 7.12.1 supports deploying FunctionBeat as an AWS Lambda service or Google Cloud Function. It responds to triggers defined for the following event sources:
