## **Concur Logging - Service Requests Procedure** 

## Overview 

The SAP Concur Logging Service team has developed prefilled service request templates to streamline the process for users requesting standard services. These templates ensure that all necessary information is provided upfront, enabling faster processing and reducing the need for follow-up. In addition to saving time for users, the structured format helps the Logging Service team efficiently track requests, gather usage statistics, and identify opportunities for future service improvements.
